A [Flarum](http://flarum.org) extension. Flarum Persona Switching

git tag v0.0.0 git push origin v0.0.0

## Installation

Install with composer:

```sh
composer require dhtml/persona:"*"
```

## Updating

```sh
composer update dhtml/persona:"*"
php flarum migrate
php flarum cache:clear
```
